## Human Approval Gateway

Open approvals appear on the dashboard home page. User actions allowed:
- Payment approval
- OTP / verification codes
- OAuth login consent
- KYC / legal confirmation

## Agent Automation (Not User Tasks)

| Function | Agent |
|----------|-------|
| New project | CEO → Project Factory |
| Platform connect | Setup Agent |
| Deploy | Deployment Agent |
| Registry updates | Orchestrator → Project Memory |
